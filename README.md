

# Volunteer, Internship & Fellowship Opportunities in India – 2026

##  Overview
This dataset compiles **15+ opportunities** for students in India, including fellowships, internships, and volunteering programs.  
It was created as part of **Task 3** for the AI Data Analytics Internship with *InAmigos Foundation*.  

## Dataset Details
The dataset includes the following columns:
- Organization / Program  
- Opportunity Type (Fellowship, Internship, Volunteering)  
- Sector / Focus Area  
- Eligibility  
- Location (India / Remote)  
- Duration (Months / Years)  
- Stipend / Support  
- Deadline  
- Application Link  
- Selection Process  
- Impact Area  
- Organization Type (NGO, Corporate, Government, UN)  
- Cohort Size (Approximate)  
- Alumni Benefits  
- Skills Gained  
- Mode (Online / Offline / Hybrid)  

##  Purpose
- To provide students with a structured resource of social-impact opportunities.  
- To practice **data organization, research, and visualization skills**.  
- To enable future analysis (e.g., stipend vs duration, sectoral distribution, deadlines timeline).  

##  Example Use Cases
- Create pivot tables to compare opportunities by sector or type.  
- Visualize stipend distribution across fellowships.  
- Map locations of opportunities (remote vs on-ground).  
- Analyze deadlines to plan applications.
